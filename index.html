Resumo do Projeto

2025 • VidaFit-Database

Este projeto demonstra criação e manipulação de dados em um banco relacional usando SQL (PostgreSQL). Inclui scripts para criação de esquema, povoamento, consultas e operações de atualização/remoção.

O que entregamos

  • Site estático pronto para GitHub Pages
  • Scripts SQL reais: CREATE, INSERT, SELECT, UPDATE, DELETE
  • README com instruções e versão para PostgreSQL (pgAdmin)

Arquivos

  • 01_create_tables.sql
  • 02_insert_data.sql
  • 03_select_queries.sql
  • 04_update_delete.sql

Como usar

  1. Abra o pgAdmin ou psql
  2. Execute em ordem: 01_create_tables.sql → 02_insert_data.sql
  3. Rode as consultas e experimente UPDATE/DELETE do script 04
