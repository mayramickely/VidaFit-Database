Início Scripts Instruções Contato

Requisitos

  • PostgreSQL 12+ (pgAdmin ou psql)
  • Ambiente para executar scripts SQL

Execução (exemplo com psql)

psql -U seu_usuario -d vida_fit -f 01_create_tables.sql
psql -U seu_usuario -d vida_fit -f 02_insert_data.sql

Boa prática

Execute os scripts em ordem e valide integridade. Use controle de versão (Git) para fazer commits frequentes.
